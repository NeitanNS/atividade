# Atividade 10 e 11

Projeto desenvolvido em React + TypeScript com consumo da API Rick and Morty.

## Funcionalidades

- Listagem de personagens consumidos da API
- Busca por nome com debounce
- Filtro por status
- Paginação entre resultados
- Favoritos com persistência em `localStorage`
- Componentização com Context API e custom hooks

## Tecnologias

- React
- TypeScript
- Context API
- Custom Hooks
- CSS

## Estrutura

```text
src/
  components/
  contexts/
  hooks/
  types/
  App.tsx
  App.css
  main.tsx
```

## Como executar

```bash
npm install
npm run dev
```

## API utilizada

[Rick and Morty API](https://rickandmortyapi.com/)
